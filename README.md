# cantuskey
CantusKey es una herramienta web gratuita creada para apoyar el servicio del canto y la música litúrgica. Permite transponer acordes de manera sencilla, elegir el tono adecuado para la asamblea y mantener la letra intacta. Pensada para coros, ministerios de música y servidores que trabajan con acordes anglosajones (A–G).
